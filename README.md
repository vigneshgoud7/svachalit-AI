# 🚀 Svachalit AI

> **AI Agents That Respond, Reason, and Act**

Svachalit AI is an Agentic AI-powered multi-channel business automation platform that enables businesses to automate customer communication, execute workflows, and manage customer interactions across multiple channels from a unified dashboard.

Unlike traditional chatbots that follow predefined rules, Svachalit AI leverages autonomous AI agents, Retrieval-Augmented Generation (RAG), and workflow orchestration to understand, reason, and take actions on behalf of businesses.

---

## 🌟 Overview

Modern businesses receive customer inquiries across multiple platforms such as WhatsApp, Instagram, Facebook, websites, emails, and voice channels.

Managing these interactions manually leads to:

* Slow response times
* Lost sales opportunities
* Increased operational costs
* Fragmented communication
* Poor customer experience

Svachalit AI solves this problem by providing intelligent AI agents capable of understanding customer intent, retrieving business knowledge, executing workflows, and collaborating with human operators when needed.

## 🚀 Core Workflow

1. Customer sends a message through any supported channel.
2. Message is received via webhooks.
3. Conversation Orchestrator manages routing and context.
4. AI Agent analyzes customer intent.
5. RAG retrieves relevant business knowledge.
6. LLM generates reasoning and response.
7. Agent executes required actions.
8. Results are synced with CRM and business systems.
9. Human agents are involved when escalation is required.

## 📦 Installation

```bash
git clone https://github.com/your-username/svachalit-ai.git

cd svachalit-ai

npm install
```

Create a `.env` file:

```env
OPENAI_API_KEY=
GEMINI_API_KEY=
DATABASE_URL=
REDIS_URL=
```

Run the application:

```bash
npm run dev
```
